# Tests

## 📂 目录简介

SDD 插件的测试代码目录，包含端到端测试 (e2e) 和其他测试文件。

## 📁 目录结构

```
tests/
├── README.md           # 本文件
└── e2e/                # 端到端测试
    ├── README.md       # e2e 测试说明
    ├── e2e-test.fixture.ts    # 测试夹具
    └── main-feature-flow.test.ts  # 主流程测试
```

## 📄 文件说明

| 文件/目录 | 类型 | 说明 |
|-----------|------|------|
| README.md | 文件 | 本导航文件 |
| e2e/ | 目录 | 端到端测试目录 |

## 🧪 运行测试

```bash
# 运行单元测试
npm test

# 运行端到端测试
npm run test:e2e
```
