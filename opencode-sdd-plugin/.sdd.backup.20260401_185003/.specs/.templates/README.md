# Templates

## 📂 目录简介

存储 SDD 规范文档的 README 模板，用于快速生成标准化的文档结构。

## 📁 目录结构

```
.templates/
├── README.md                 # 本文件
├── feature-README.md         # Feature 级别 README 模板
└── subfeature-README.md      # Sub-Feature 级别 README 模板
```

## 📄 文件说明

| 文件/目录 | 类型 | 说明 |
|-----------|------|------|
| README.md | 文件 | 本导航文件 |
| feature-README.md | 文件 | Feature 级别 README 模板 |
| subfeature-README.md | 文件 | Sub-Feature 级别 README 模板 |

## 📖 模板用途

### feature-README.md
用于标准 Feature 目录的 README，包含：
- Feature 状态信息
- 文件列表
- 概述和导航

### subfeature-README.md
用于 Sub-Feature 目录的 README，包含：
- 子 Feature 详细说明
- 与父 Feature 的关系
- 任务分解
